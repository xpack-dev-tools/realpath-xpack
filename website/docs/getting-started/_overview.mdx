The main target of this project is macOS, which does not include this
GNU utility.

`realpath` is an utility to compute the real path of a filesystem path;
it is part of the [GNU Core Utilities](https://www.gnu.org/software/coreutils/).

**GNU Core Utilities** is an open source project hosted on
[Savannah](https://savannah.gnu.org/projects/coreutils).
